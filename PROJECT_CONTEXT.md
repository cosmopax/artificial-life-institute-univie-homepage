# Project Context

This is a custom static site generator built with Python, hosted on University of Vienna webspace.

- **Source**: `content/` (JSON, CSV, MD/TXT)
- **Generator**: `tools/build.py`
- **Output**: `site/`
- **Deploy**: `deploy.sh` (rsyncs to mounted webspace `/Users/cosmopax/WebspaceMount/a0402554/artificiat27/html`)
- **Live URL**: <https://artificial-life-institute.univie.ac.at/>

Dependencies are minimal (standard library).
